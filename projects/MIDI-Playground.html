MIDI Playground

Project Description

MIDI Playground is a browser-based MIDI recording and playback studio featuring a modern dark theme UI. It allows users to record MIDI input from any connected controller via the Web MIDI API, play polyphonic tones using Tone.js, and export recordings as standard Format 0 .mid files.

The project avoids all external MIDI libraries entirely. MIDI files are written from scratch using hand-crafted binary formatting with custom VLQ encoding, and exported .mid files are parsed byte-by-byte during playback — translating raw MIDI data back into Tone.js scheduling triggers. Features include track arm, instrument selection (polyphonic synth or Salamander piano sampler), playlist management, and export to downloadable MIDI files.

Technologies Used

  • JavaScript (ES2020+)
  • HTML / CSS
  • Tone.js (audio synthesis only)
  • Web MIDI API
  • Web Audio API
