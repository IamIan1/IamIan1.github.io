PiMemorizer

Project Description

PiMemorizer is a simple Python application that aims to maximize the efficiency at which the user memorizes digits of Pi. It has two distinct modes. The first is an 'endless' mode, which allows users to enter as many digits as they can, offering corrections when necessary. This mode is best for the initial learning phase. The second mode forces the user to restart from the beginning whenever they make a mistake, which enhances the memorization process via repetition while also offering corrections for incorrect digits.

In either mode, it will save progress across sessions in a json file that's loaded when the app is started, and updated when it's closed. Additionally, it constantly displays the number of digits entered.
