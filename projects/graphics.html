Computer Graphics

Project Description

In this computer graphics project, I developed a Static Ray Tracer using C++ that not only featured multiple light sources, shadows, and complex scene rendering but also introduced support for reflective objects. This project showcased my proficiency in ray tracing and my ability to handle intricate lighting interactions and material properties, including reflections.

Key Achievements and Contributions:

Comprehensive Ray Tracing Engine: I engineered a comprehensive ray tracing engine that handled ray-object intersections, multi-light source illumination, shadows, and reflections. This robust engine formed the foundation for rendering complex and visually captivating scenes.

Multiple Light Sources: The ray tracer seamlessly integrated multiple light sources, allowing for the accurate computation of lighting contributions from each source. This enhanced the realism of the rendered scenes.

Realistic Shadows: I implemented precise shadow casting by employing shadow rays to determine whether objects were occluded from the perspective of each light source. This ensured lifelike and accurate shadow effects.

Reflective Objects: The project introduced support for reflective materials, enabling objects to interact with their surroundings by reflecting light rays. This feature added depth and complexity to the rendered scenes, showcasing mirrored surfaces and intricate light interactions.
